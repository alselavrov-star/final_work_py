Алексей Лавров_Итоговая работа дисциплине "Программирование на python"

В данной работе реализован инструмент для параллельной обработки 
логов утилиты suricata и анализа доменных имен, IP адресов и файлов 
на предмет наличия вредоносного ПО с помощью virustotal.
Проект разбит на пять модулей для соблюдения принципов ООП
Каждый из модулей выполняет отдельную функцию, а именно:
suricata_processor - обработка логов suricata;
virustotal_processor - проверка доменных имен, IP адресов и файлов 
на предмет наличия вредоносного ПО с помощью virustotal API;
telegram_notifier - отправка сообщений в телеграм;
plot_generator - создание и сохранение графиков;
main - "тело" проекта, в котором описывается взаимодействие модулей между собой.
Процесс обработки логов и проверки с помощью virustotal выделены в параллельные потоки, 
так как у двух этих сревисов принципиально различается логика работы. 
Если логи пишутся динамически, то обработка через virustotal происходит по запросу от пользователя. 
Таким образом, имеет смысл разделить эти процессы. 
Собранные в результате работы программы данные сохраняются в папку ./reports, для suricata -  в формате csv, 
для virustotal - в формате json. В эту же директорию сохраняются графики, созданные на основе собранных данных.
